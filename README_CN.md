对象的生命周期一直是开发者所需要关心的，该 Demo 测试了 UIViewController & UIView 的生命周期方法。您可以下载并使用 Xcode 运行，通过控制台输出来观察结果。

如果项目中有任何 Bug 或改进，您都可以直接发起 Issue，定虚心学习。谢谢！
